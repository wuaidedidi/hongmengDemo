后端API测试示例
==================

服务器地址: http://localhost:8080/api

1. 登录大白兔用户
==================
POST /auth/login
Content-Type: application/json

{
  "username": "大白兔",
  "password": "7758258"
}

返回示例:
{
  "token": "eyJhbGciOiJIUzI1NiJ9...",
  "username": "大白兔"
}

2. 获取待办事项列表
==================
GET /todos
Authorization: Bearer YOUR_TOKEN_HERE

3. 添加待办事项
==================
POST /todos
Authorization: Bearer YOUR_TOKEN_HERE
Content-Type: application/json

{
  "title": "学习HarmonyOS开发",
  "description": "完成第一个应用开发",
  "priority": "HIGH"
}

{
  "title": "写技术博客",
  "description": "分享开发经验",
  "priority": "MEDIUM"
}

{
  "title": "锻炼身体",
  "description": "每天跑步30分钟",
  "priority": "LOW"
}

4. 获取待办合集列表
==================
GET /todo-collections
Authorization: Bearer YOUR_TOKEN_HERE

5. 添加待办合集
==================
POST /todo-collections
Authorization: Bearer YOUR_TOKEN_HERE
Content-Type: application/json

{
  "name": "学习计划",
  "description": "技术学习相关的任务"
}
